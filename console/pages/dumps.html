About

These are dumps of the MySQL database of TTNmon. They contain the tables gateways and packets. The tables authorizations and devices as well as the preprocessed tables are not included.

The table packets contains all packets. Each packet has a unique packet id and is related to a device pseudonym. It holds information about the spreading factor, frequency, time and node position (if available).

The table gateways contains all gateways which received a packet. Each row is related to the packet id. It holds information about the link quality to a gateway like SNR or RSSI
