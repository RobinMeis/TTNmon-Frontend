About

These are dumps of the MySQL database of TTNmon. They contain the tables gateways, packets, networks and gateway_list. The tables authorizations and devices as well as the preprocessed tables are not included.

The table packets contains all packets. Each packet has a unique packet id and is related to a device pseudonym. It holds information about the spreading factor, frequency, time and node position (if available).

The table gateways contains all gateways which received a packet. Each row is related to the packet id. It stores information about the link quality to a gateway like SNR or RSSI.

The table networks contains a list of all currently known network IDs and prefixes used for LoRaWAN. This list might be incomplete. Please contact blog@smartnoob.de if you are missing an ID.

The table gateway_list contains details for all gateways. It is generated from existing data.
